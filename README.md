# Skybridge Digital

Welcome to the official repository for the **Skybridge Digital** website. We are a premier web development and digital marketing agency dedicated to creating exceptional digital experiences.

## Project Overview

This project is the main website for Skybridge Digital, showcasing our services, portfolio, and brand identity. It is built with a modern tech stack to ensure performance, scalability, and a premium user experience.

### Tech Stack

- **Framework:** [React](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [shadcn/ui](https://ui.shadcn.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)

## getting Started

To run this project locally, follow these steps:

### Prerequisites

Ensure you have Node.js and npm installed on your machine.

### Installation

1.  **Clone the repository:**
    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  **Navigate to the project directory:**
    ```sh
    cd Skybridge_website
    ```

3.  **Install dependencies:**
    ```sh
    npm install
    ```

4.  **Start the development server:**
    ```sh
    npm run dev
    ```

   The application will be available at `http://localhost:8080` (or the port shown in your terminal).

## Deployment

The project is optimized for deployment on modern web hosting platforms like Vercel, Netlify, or similar services that support Vite-based React applications.
